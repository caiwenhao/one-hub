概览（面向普通用户）

- 入口地址：`/panel`（登录后进入你的控制台主页）。
- 你能做什么：查看使用情况、下载账单、管理 API 密钥、查看可用模型、管理个人信息。
- 布局结构：顶部操作（主题/语言/密度/个人抽屉）+ 左侧菜单 + 主内容区。

主要入口与常用操作

- 仪表盘（/panel/dashboard）
  - 查看当日请求、消费、Token 用量与近 7 日趋势。
  - 关注“状态”页的可用率与服务状态（若已启用）。
- 日志（/panel/log）
  - 支持按时间、模型、状态等筛选查看你的调用记录。
  - 可导出 CSV（仅包含你自己的数据）。
- 月度账单（/panel/invoice）
  - 选择月份查看账单概要与明细，支持跳转到某日详情。
- 可用模型（/panel/model_price）
  - 查看当前可用模型与价格信息，便于在代码中选择与对齐成本。
- 令牌（/panel/token）
  - 创建/禁用/重新命名你的 API Key，注意妥善保管密钥。
- 个人设置（/panel/profile）
  - 查看额度与已用量，访问快捷入口，管理个人信息。

小贴士

- 看不到某些菜单？那通常是管理员功能，普通用户不需要关心。
- 导出失败或没有数据？请检查筛选条件与时间范围，或稍后刷新重试。
- 切换 UI 密度可更适合你的屏幕与数据密度需求（右上角“密度”按钮）。

常见问题（FAQ）

- 登录后没有进入控制台？访问 `https://你的域名/panel` 或点击网站右上角“控制台”。
- 找不到 API Key？在“令牌”页面创建；若受限，请联系管理员开通权限。
- 账单金额不一致？账单按平台结算口径统计，若需发票或对账单，请联系管理员。
## 导航概览

- 首页：关键指标与近期动态总览
- 资源/项目：资源管理与配置项
- 开发者/API：密钥、Webhook、沙箱环境
- 日志与监控：调用记录、错误与告警
- 账号与计费：订阅、额度、发票

## 常用操作

- 创建与管理 API 密钥（开发者/API）
- 配置渠道、回调地址与白名单
- 查看调用日志、导出失败记录
- 设置告警阈值与通知方式

## 权限与团队

- 角色：拥有者/管理员/成员（最小必要权限原则）
- 邀请成员：通过邮箱邀请加入团队
- 审计：关键操作在日志中可追溯

<Note>如需更细粒度的权限模型，请联系支持。</Note>
