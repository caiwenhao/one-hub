## 导航概览

- 首页：关键指标与近期动态总览
- 资源/项目：资源管理与配置项
- 开发者/API：密钥、Webhook、沙箱环境
- 日志与监控：调用记录、错误与告警
- 账号与计费：订阅、额度、发票

## 常用操作

- 创建与管理 API 密钥（开发者/API）
- 配置渠道、回调地址与白名单
- 查看调用日志、导出失败记录
- 设置告警阈值与通知方式

## 权限与团队

- 角色：拥有者/管理员/成员（最小必要权限原则）
- 邀请成员：通过邮箱邀请加入团队
- 审计：关键操作在日志中可追溯

<Note>如需更细粒度的权限模型，请联系支持。</Note>
