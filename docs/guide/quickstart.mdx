## 第一步：注册与登录

<AccordionGroup>
  <Accordion icon="user" title="创建账号并登录">
    使用邮箱或第三方登录创建 kapon 账号，完成邮箱验证后登录控制台。
  </Accordion>
  <Accordion icon="users" title="创建或加入团队（可选）">
    新建团队并邀请成员，或加入现有团队以共享资源与配额。
  </Accordion>
</AccordionGroup>

## 第二步：基础配置

<AccordionGroup>
  <Accordion icon="gauge" title="控制台快速设置">
    普通用户需完成：创建 API 密钥（令牌），并在本地/服务器配置使用；渠道与回调等平台级设置通常由管理员完成。详情见《控制台指南》（guide/console）。
  </Accordion>
  <Accordion icon="key" title="创建 API 密钥">
    在“开发者/API”中生成并保存密钥，用于服务端调用。
  </Accordion>
</AccordionGroup>

## 第三步：首次调用

<AccordionGroup>
  <Accordion icon="terminal" title="调用示例">
    使用后端环境调用 API，示例与认证方式见【/api/overview】与【/api/authentication】。
    
    <Tip>请仅在服务端保存与使用密钥，避免泄露。</Tip>
  </Accordion>
  <Accordion icon="check" title="验证结果">
    在控制台查看调用日志或结果页面，确认已成功接入。
  </Accordion>
</AccordionGroup>
