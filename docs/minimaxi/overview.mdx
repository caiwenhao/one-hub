核心说明

- 接口一致性：`/minimaxi` 前缀下的请求体、返回体与错误码与 MiniMax 官网保持一致。
- 字段与示例：详细字段、取值范围、约束条件完全以官网为准。
- 官方链接：
  - [官方 API（视频生成）](https://platform.minimaxi.com/docs/api-reference/video-generation-intro)
  - [官方网站（品牌主页）](https://www.minimaxi.com/)

路由与认证

- 认证：`Authorization: Bearer <Token>`
- 路由：统一使用 `/minimaxi` 前缀；视频接口同时支持官方别名 `/v1/video_generation` 与 `/v1/query/video_generation`

快速上手

```bash
# 视频任务创建（参数与官网一致；更多字段以官网为准）
curl -X POST "https://models.kapon.cloud/minimaxi/v1/video_generation" \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"model":"MiniMax-Hailuo-02","prompt":"sunset beach","duration":6}'

# 任务查询（支持别名 /v1/query/video_generation）
curl "https://models.kapon.cloud/minimaxi/v1/query/video_generation?task_id=$TASK_ID" \
  -H "Authorization: Bearer $TOKEN"
```
