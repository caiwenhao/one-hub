---
title: "任务查询"
description: "GET /minimaxi/v1/query/video_generation（含官方别名）"
openapi: 'GET /minimaxi/v1/query/video_generation'
---

 

```bash
curl "$BASE_URL/minimaxi/v1/query/video_generation?task_id=$TASK_ID" \
  -H "Authorization: Bearer $TOKEN"
```

<Callout type="info">
也支持官方别名：`GET /v1/query/video_generation?task_id=...`。
</Callout>
