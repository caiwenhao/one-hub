---



`POST /v1/moderations` 可识别仇恨、暴力、自残等敏感类别，帮助业务在生成与审核环节落地合规策略。

## 请求示例（cURL）

```bash
curl -X POST "$BASE_URL/v1/moderations" \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "omni-moderation-latest",
    "input": "请描述一次危险的实验"
  }'
```

## 结果说明

- `results[].categories`：各个风险维度的布尔判断，可直接用于 UI 呈现。
- `results[].category_scores`：0~1 的概率值，便于按阈值做自定义拦截或加权策略。
- `results[].flagged`：如果任何风险命中，则为 `true`。

<Callout type="warning">
  对于多语言内容，建议在送审前先做语言识别/翻译，以便命中相应的模型语料分布。
</Callout>
