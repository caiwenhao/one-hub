title: "Videos"
description: "通过 kapon 代理调用 OpenAI Sora 视频接口，管理生成任务与素材。"
openapi: 'POST /v1/videos'
---

 

## 创建视频任务

  ```bash
  curl -X POST "$BASE_URL/v1/videos" \
    -H "Authorization: Bearer $TOKEN" \
    -H "Content-Type: application/json" \
    -d '{
      "model": "sora-2",
      "prompt": "一个无人机从海滩升空拍摄夕阳",
      "seconds": 8,
      "size": "1280x720"
    }'
  ```

## 查询与删除

  ```bash
  curl "$BASE_URL/v1/videos/$VIDEO_ID" \
    -H "Authorization: Bearer $TOKEN"

  curl -X DELETE "$BASE_URL/v1/videos/$VIDEO_ID" \
    -H "Authorization: Bearer $TOKEN"
  ```

## 下载与 Remix

  ```bash
  curl -L "$BASE_URL/v1/videos/$VIDEO_ID/content" \
    -H "Authorization: Bearer $TOKEN" \
    --output "$VIDEO_ID.mp4"

  curl -X POST "$BASE_URL/v1/videos/$VIDEO_ID/remix" \
    -H "Authorization: Bearer $TOKEN" \
    -H "Content-Type: application/json" \
    -d '{"prompt": "在结尾添加品牌 Logo"}'
  ```

<Callout type="info">
  kapon 会对不同供应商的返回结构进行统一封装，`status` 字段取值为 `queued`、`processing`、`succeeded`、`failed` 等；`assets` 数组中包含各个质量/角度的素材链接。
</Callout>
