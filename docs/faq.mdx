# 常见问题（FAQ）

## 如何本地预览文档？

- 安装 Mintlify CLI：`npm i -g mint`
- 在 `docs/` 目录下运行：`mintlify dev`
- 浏览器打开本地地址查看预览。

## 文档导航如何配置？

- 修改 `docs/docs.json` 中的 `navigation` 字段。
- 新增页面时，确保对应的 `.mdx` 文件路径与导航中 `pages` 一致。

## 如何贡献或修正文档？

- 在对应 `.mdx` 页面直接修改并提交合并请求。
- 保持标题、描述与导航一致，避免重复内容（遵循 DRY ）。

## 有问题如何联系？

- 通过站点右上角“支持”链接，或在仓库提交 Issue。
